安装hexo

  • npm install -g hexo
  • hexo init
  • hexo generate（hexo g也可以）

  • hexo server（hexo s也可以）

    • 默认端口4000,可能被福昕占用
    • 换用端口5000:

      hexo s -p 5000

  • 更换皮肤themes

  • 配置github

    type: git
    repo: https://github.com/yuhaihua200/yuhaihua200.github.io.git

常用命令

  • hexo clean #清除缓存文件夹public
  • npm install hexo-deployer-git –save #安装node_modules
  • hexo new”postName”(简写 hexo n) #新建文章
  • hexo new page”pageName” #新建页面
