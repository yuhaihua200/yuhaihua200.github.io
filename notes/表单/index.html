表单总结

form

表单都是以此开始

input

必须的:

  • type
    • text:文本输入
    • submit:创建提交按钮
    • radio:单选按钮
      • value:选择按钮必须提供此值
    • checkbox:复选框
      • value:选择按钮必须提供此值
    • number:只能输入数字
      • min:设置最小值
      • max:设置最大值
    • range:显示一个滚动条(类似number,但无输入)
      • min
      • max
      • step:步长
    • color:颜色选择
    • date:日历选择
    • 以下三个在电脑端看不出区别:
      • email:在移动端得到方便输入email的定制键盘
      • tel:在移动端得到方便输入电话号码的定制键盘
      • url:在移动端得到方便输入url地址的定制键盘
  • name:为元素命名

textarea

创建一个多行的文本区(双闭合标签)

  • name:命名
  • rows:文本区高度
  • cols:文本区宽度
    可以预制初始文本

select

菜单控件,通常与option标签结合使用

  • name:必须命名

option

下拉菜单选择项,必须与select结合使用

  • value:必须有

label

label标签为 input 元素定义标注（标记）。
label 元素不会向用户呈现任何特殊效果。不过，它为鼠标用户改进了可用性。如果您在 label 元素内点击文本，就会触发此控件。(点击label的文字,就会触发控件发生焦点)就是说，当用户选择该标签时，浏览器就会自动将焦点转到和标签相关的表单控件上。
label 标签的 for 属性应当与相关元素的 id 属性相同。(也就是说label中for值要对应input的id值,否则不起作用)

###button和submit区别

  • button:一个按钮
  • submit:提交表单
